## 프로젝트 개요

고밀도 도시 환경에서 전동 이륜차의 무단 엘리베이터 반입은 리튬 배터리 불안정으로 인한 심각한 화재 위험을 초래합니다. 건물 관리자는 핵심적인 안전 요구사항에 직면합니다: <Gradient from="red" to="orange"> 문이 닫히기 전에 전동 이륜차를 차단하라. </Gradient>

**MOTA TECHLINK**은 금지 객체를 실시간으로 감지하는 엔드투엔드 **엣지 AIoT 솔루션**을 설계했습니다. 전통적인 데이터 수집에서 **생성형 AI(GenAI) 합성 데이터 생성**으로 전환하여 데이터 부족 병목현상을 해소하고, 고도로 최적화된 프라이버시 준수 모델을 임베디드 하드웨어에 직접 배포했습니다.

## 도전 과제: 안전 vs. 데이터 제약

엘리베이터라는 밀폐된 동적 환경에서 견고한 감지 시스템을 개발하는 것은 독특한 엔지니어링 난제를 제시합니다:

1.  **데이터 부족 및 프라이버시 (GDPR):** 실제 학습 데이터 수집은 법적으로 복잡합니다. 엘리베이터에서 거주자의 이미지를 촬영하는 것은 상당한 프라이버시 우려(GDPR/PIPL)를 야기하며, 이륜차 사진을 연출하는 것은 노동 집약적입니다.
2.  **"롱테일" 코너 케이스:** 모델은 예측 불가능한 시나리오—혼잡한 엘리베이터, 다양한 조명 조건, 부분적으로 가려진 이륜차, 혼동 객체(예: 휠체어 vs. 이륜차)—에서 종종 실패합니다.
3.  **엣지 제약:** 솔루션은 **오프라인**에서 **초저지연**으로 작동해야 합니다. 비디오 스트림을 클라우드로 전송하여 추론하면 허용할 수 없는 지연과 인터넷 연결 의존성이 발생합니다. 시스템은 저전력, 비용 효율적인 NPU 하드웨어에서 실행되어야 합니다.

## 솔루션: GenAI 우선 워크플로

컴퓨터 비전 개발 라이프사이클을 재구상했습니다. 데이터를 '수집'하는 대신 '생성'했습니다. 이 접근 방식은 모델 견고성을 향상시키면서 개발 주기를 수개월에서 수일로 단축했습니다.

### 1. 합성 데이터 생성 (GenAI)
멀티모달 대규모 언어 모델(LLM)과 확산 모델을 활용하여 사실적인 장면을 생성했습니다.
![GenAI 샘플](/images/showcase/Edge-Computing/GenAI-Samples.webp)
* **도메인 랜덤화:** 프롬프트를 사용하여 조명, 엘리베이터 텍스처(거울, 스틸), 각도를 다양화했습니다.
* **설계 단계부터 프라이버시 보호:** 학습 데이터의 인물이 AI로 생성되므로 프라이버시나 동의 문제가 없습니다.
* **엣지 케이스 커버리지:** 우비로 덮인 이륜차나 극단적인 탑다운 각도에서 촬영한 것과 같은 희귀 시나리오를 합성으로 생성했습니다.
 

* **산업 검증:** <GifVideo src="/videos/GenAI_compressed.mp4" align="left"  width="30%" />이 데이터 중심 전략은 <Gradient from="red" to="orange">실제 세계에서 통계적으로 포착하기 어려운 희귀 "코너 케이스"를 위해 생성형 시뮬레이션과 합성 데이터를 활용하여 <a href="https://bernardmarr.com/how-tesla-is-using-artificial-intelligence-to-create-the-autonomous-cars-of-the-future/">오토파일럿 시스템을 훈련</a>하는 Tesla의 첨단 방법론</Gradient>과 일치합니다. 이는 합성 데이터가 단순한 우회책이 아니라 견고성을 향한 우수한 경로임을 증명합니다.

### 2. 자동 어노테이션 파이프라인
수동 라벨링은 AI 학습의 병목입니다. <ColorText color="green">Segment Anything Model</ColorText> [SAM3](https://github.com/facebookresearch/sam3)을 파이프라인에 통합했습니다.
* **자동 세그멘테이션:** SAM이 합성 데이터셋에 대해 픽셀 단위 완벽한 바운딩 박스와 세그멘테이션 마스크를 자동 생성했습니다.
* **효율성:** 이 자동화로 데이터 준비 시간을 **90% 단축**하면서 인간 어노테이터를 능가하는 일관된 라벨 품질을 보장했습니다.
![](/images/showcase/Edge-Computing/SAM.webp)

## 기술 실행: 엣지 최적화

모델을 훈련하는 것은 절반에 불과합니다. 저전력 칩에 배포하는 것이 MOTA TECHLINK이 탁월한 분야입니다.

### NPU 양자화 및 압축
비용 효율적인 IoT 하드웨어(RISC-V/ARM 아키텍처)에서 실행하기 위해 무거운 GPU 모델을 사용할 수 없었습니다.
* **PTQ (학습 후 양자화):** YOLO 기반 아키텍처를 FP32(32비트 부동소수점)에서 **INT8(8비트 정수)**로 압축했습니다.
* **결과:** 모델 크기를 **75% 축소**하고 추론 속도를 **300% 향상**시켜 제한된 RAM의 엣지 NPU에서 원활하게 실행되도록 했습니다.
![GenAI 샘플](/images/showcase/Edge-Computing/Quantization.webp)

### 실시간 로직 및 제어
시스템은 수동적인 카메라가 아닌 능동적인 컨트롤러로 작동합니다.
* **지연 시간:** 100ms 미만의 추론 시간.
* **동작:** 높은 신뢰도(>0.85)로 대상을 감지하면 엣지 디바이스가 GPIO 릴레이를 트리거하여 **엘리베이터 문을 열린 상태로 유지**하고 음성 경고를 재생합니다("금지 객체가 감지되었습니다").
* **안전 루프:** 객체가 제거될 때까지 엘리베이터가 움직이지 않아 물리적으로 위험을 방지합니다.
![GenAI 샘플](/images/showcase/Edge-Computing/CV-Controls.webp)

## 영향 및 결과

* **시장 출시 속도:** 모델 개발 시간을 **3개월에서 2주로** 단축.
* **프라이버시 준수:** 합성 데이터를 통해 데이터 프라이버시 규정 100% 준수.
* **비용 효율성:** 고가의 산업용 PC 대신 저비용 엣지 칩 사용 가능.
* **정확도:** 파일럿 배포에서 **99.2%** 감지 정확도 달성, 휠체어(허용)와 전동 이륜차(금지)를 성공적으로 구별.

## 기술 스택

* **학습 프레임워크:** PyTorch, Ultralytics YOLO
* **데이터 생성:** Stable Diffusion, ControlNet
* **어노테이션:** Meta AI SAM (Segment Anything Model)
* **엣지 추론:** TensorRT / TFLite Micro / RKNN (Rockchip NPU)
* **하드웨어:** 카메라 모듈이 장착된 커스텀 RISC-V / ARM 임베디드 보드
