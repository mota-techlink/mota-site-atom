### 課題：高齢者の安全・つながり・孤独解消を一つのデバイスで

移動に困難を抱える高齢者にとって、杖はすでに欠かせないツールです。しかし物理的サポートにとどまらず、この層は従来の補助具が解決できない三つの複合課題に直面しています。

*   **緊急時の位置確認：** 高齢者が転倒したり、不慣れな屋外環境で迷子になったりしたとき、家族や介護者が迅速に場所を特定できる手段がありません。
*   **デジタルデバイド：** 多くの高齢者はスマートフォンを操作するための手先の器用さ、視力、またはデジタルリテラシーが不足しています。音声は最も自然なインターフェイスですが、常に手に持ち、常時オンのデバイスが必要です。
*   **認知的エンゲージメントと感情的サポート：** 孤独感と認知機能の低下は高齢者にとって深刻な健康リスクです。受動的なデバイスは会話の関与や認知刺激を一切提供しません。

課題は、ユーザーがすでに手に持っているデバイス——杖——にこれらすべての機能を組み込みながら、エンドユーザーに技術的複雑さを追加しないことでした。

### ソリューション：杖に乗る MOTA AI Agent プラットフォーム


#### ハードウェアアーキテクチャ

デバイスは専用ハードウェアと MOTA のクラウドエッジソフトウェアレイヤーを組み合わせています：

| コンポーネント | 機能 |
|---|---|
| **音声チップ** | デバイス上での音声キャプチャ・再生 |
| **利尔达 4G モジュール** | 常時オンのモバイル通信、クラウド推論に対応 |
| **北斗測位チップ** | リアルタイム衛星 GPS 位置追跡 |
| **AI ボタン** | 長押しで話す一ボタン操作——画面も APP も不要 |

#### インタラクション：プッシュ・トゥ・トーク

ユーザー体験全体が単一の物理ジェスチャーに集約され、高齢者の運動能力に最適化されています：

1. AI ボタンを**長押し** → マイクが起動し、ユーザーが自然に話す。
2. ボタンを**離す** → 音声入力が 4G 経由でクラウド LLM にストリーミング送信される。
3. LLM がクエリを処理し、合成音声として応答をストリーミング返信。
4. ユーザーは内蔵スピーカーから応答を聞く。

**割り込みサポート：** LLM が応答を再生中にユーザーが再度長押しすると、再生が即座に停止し、新しいクエリの受付準備が整います。

#### LLM 統合：中国移動大規模モデル

MOTA の Agent プラットフォームに**中国移動 LLM** コネクターを新規追加。エンタープライズはエージェント設定でこれを基盤モデルとして選択できます：

*   **自然な会話**——高齢者に多い省略的・口語的な表現を理解します。
*   **垂直ドメイン知識**——健康リマインダー、服薬スケジュール、地域の天気など。
*   **セッション内マルチターンメモリ**——自然な流れで会話を続けられます。

#### Agent API レイヤー

MOTA は南京立奇のハードウェアプラットフォーム向けに二つのクリーンな統合インターフェースを構築しました：

*   **ASR（音声認識）：** マイクが収録した生音声ストリームを受け取り、LLM 推論用テキストを返します。
*   **TTS（音声合成）：** LLM のテキスト応答を受け取り、合成音声ストリームと再生コマンドをケインのスピーカーへ返します。

明確な API 境界によりハードウェアチームと AI プラットフォームチームが独立して進化でき、統合の複雑さを低減し、将来のアップデートを加速します。

### 方案の価値

音声 AI、衛星測位、LLM エージェントを日常使用の医療補助具に統合することで、三つの次元で測定可能な価値を実現しています：

*   **安全確保：** 北斗 GPS が家族や介護者へのリアルタイム位置共有を可能にし、緊急対応の迅速化につながります。
*   **バリアフリー：** 操作は一つのボタンを押すだけ——スマートフォンも APP も学習コストも不要です。
*   **感情的なつながり：** 自然言語 LLM 会話が、独居や低交流環境にいる高齢者に認知刺激と感情的なつながりを提供します。
