## 개요

생성형 AI 시대에 기업은 특정 도메인 지식을 이해하고, 데이터 프라이버시를 보장하며, 낮은 지연 시간으로 운영되는
<Gradient from="red" to="orange">지능형 어시스턴트</Gradient>가 필요합니다. 표준 "래퍼" 솔루션은 높은 API 비용, 느린 응답 시간, 데이터 환각 문제를 겪는 경우가 많습니다.

**MOTA TECHLINK**은 **검색 증강 생성(RAG)**을 활용하여 프라이빗 기업 데이터를 기반으로 정확하고 맥락을 인식하는 답변을 제공하는 풀스택 엣지 네이티브 AI 솔루션을 설계했습니다. 전통적인 무거운 서버를 우회하고 Cloudflare의 글로벌 엣지 네트워크를 활용하여 비할 데 없는 성능과 비용 효율성을 달성했습니다.

## 도전 과제

정적 사이트 아키텍처(Hugo)에서 프로덕션 레벨 AI 챗봇을 구축하는 것은 몇 가지 독특한 엔지니어링 과제를 제시합니다:

1.  **엣지 런타임 호환성:** 표준 Node.js 인증 라이브러리(Auth.js 등)는 V8 엣지 런타임에서 사용할 수 없는 API에 의존합니다.
2.  **상태 관리:** 서버리스, 무상태 환경에서 레이스 컨디션 없이 OAuth 2.0 흐름과 대화 기록을 처리합니다.
3.  **실시간 경험:** 비동기 데이터베이스 쓰기를 통한 히스토리 감사를 동시에 수행하면서 부드러운 타자기 스타일의 스트리밍 응답(SSE)을 구현합니다.
4.  **비용 및 정확도:** 고품질 LLM 추론의 필요성과 엄격한 운영 비용 관리 사이의 균형을 맞춥니다.

## 솔루션: 엣지 네이티브 아키텍처

 >단순한 API 호출을 넘어 Cloudflare의 최신 스택을 사용하여 견고한 **서버리스 AI 게이트웨이**를 구축했습니다.

![](/images/showcase/chatbot/edge-solution.webp)

### 시스템 아키텍처

* **프론트엔드:** Hugo (정적) + 바닐라 JS (SSE 파서 & 마크다운 렌더러).
* **엣지 API 게이트웨이:** Cloudflare Pages Functions (V8 런타임).
* **인증:** **Workers KV**로 보호되는 커스텀 네이티브 OAuth 2.0 흐름 (Google).
* **지식 베이스 (RAG):** **Cloudflare Vectorize** (벡터 DB) + **Workers AI** (임베딩 모델).
* **추론 엔진:** Workers AI에서 실행되는 **Llama 3** (엣지 GPU).
* **영속성:** 사용자 프로필과 채팅 기록을 위한 **Cloudflare D1** (서버리스 SQL).

### 주요 기능

#### 1. 검색 증강 생성 (RAG)
범용 챗봇과 달리 우리 시스템은 답변하기 전에 MOTA TECHLINK의 독점 문서를 "읽습니다". **AutoRAG** 워크플로를 활용하여 관련 컨텍스트를 임베딩, 인덱싱, 검색하여 답변의 정확성을 보장하고 환각을 방지합니다.

#### 2. 네이티브 OAuth 2.0 구현
엣지 런타임 제한을 극복하기 위해 경량의 네이티브 OAuth 인증 흐름을 설계했습니다.
* **보안:** **Workers KV**를 사용한 상태 검증으로 CSRF 공격을 방지합니다.
* **세션 관리:** **D1**에 저장된 안전한 HTTP-only JWT 세션으로 10ms 미만의 지연 시간으로 엣지에서 사용자 신원을 검증합니다.

#### 3. 실시간 스트리밍 및 히스토리
커스텀 **Server-Sent Events (SSE)** 프로토콜을 구현했습니다.
* **논블로킹 I/O:** 시스템은 즉각적인 피드백을 위해 AI 응답을 바이트 단위로 사용자에게 스트리밍합니다.
* **비동기 로깅:** `Response.tee()`와 `context.waitUntil`을 활용하여 사용자 경험에 지연을 추가하지 않으면서 대화 기록을 비동기적으로 D1 데이터베이스에 유지합니다.

## 기술적 하이라이트

### 엣지 호환성 퍼즐 해결
표준 인증 라이브러리는 엣지 환경에서 실패했습니다. 표준 `fetch` API를 사용하여 토큰 교환과 OpenID Connect 사용자 정보 가져오기를 수동으로 처리하도록 OAuth 콜백 로직을 재작성하여 Cloudflare V8 런타임과 100% 호환성을 보장했습니다.

### 비용 및 성능 최적화
"지식 주입" 접근법(프롬프트에 모든 문서 전송)에서 RAG 아키텍처로 전환하여 **입력 토큰 소비를 90% 감소**시켰습니다. 더 나아가 Cloudflare Workers AI를 활용하여 엣지 GPU에서 추론을 실행함으로써 비싼 상시 가동 GPU 서버의 필요성을 제거했습니다.

## 지원을 넘어: 평생 학습 자산
![](/images/showcase/chatbot/lifegrowth.webp)

이 기술의 진정한 가치는 **연중무휴 다국어 고객 지원** 제공을 넘어섭니다. 그 핵심 강점은 **진화 능력**에 있습니다:

* **지속적 개선 루프:** 모든 상호작용은 D1 데이터베이스에 안전하게 기록됩니다.
* **능동 학습:** 관리자는 대화 로그를 검토하고, AI 응답의 품질을 평가하며, 지식 베이스의 격차를 식별할 수 있습니다.
* **평생 성장:** 수정된 정보와 새로운 비즈니스 사례를 RAG 시스템에 지속적으로 피드백함으로써 AI의 지식 베이스가 유기적으로 성장합니다. 단순히 질문에 답변하는 것이 아니라 시간이 지남에 따라 더 똑똑해지고, 더 정확해지며, 비즈니스 목표에 더 부합하게 됩니다.

## 통합 및 서비스

이것은 단순한 쇼케이스가 아닌 배포 준비가 된 솔루션입니다.

**이 엣지 네이티브 AI 기술을 기존 웹사이트나 플랫폼에 통합하는 것을 기꺼이 도와드리겠습니다.** 정적 사이트, WordPress 블로그, 커스텀 엔터프라이즈 애플리케이션 등 어떤 환경에서든 당사의 모듈형 아키텍처는 원활한 임베딩을 가능하게 합니다. 정적 콘텐츠를 인터랙티브하고 지능적인 지식 허브로 변환하는 것을 도와드리겠습니다.
