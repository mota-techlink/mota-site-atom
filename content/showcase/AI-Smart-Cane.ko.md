### 과제: 안전, 연결성, 동반자 관계를 하나의 디바이스로

이동에 어려움을 겪는 고령자에게 지팡이는 이미 없어서는 안 될 도구입니다. 하지만 물리적 지지를 넘어 이 계층은 전통적인 보조 기기가 해결할 수 없는 세 가지 복합적인 과제에 직면해 있습니다.

*   **응급 위치 파악:** 노인이 낙상하거나 익숙하지 않은 야외 환경에서 길을 잃었을 때, 가족과 돌봄 제공자가 신속하게 위치를 찾을 수 있는 신뢰할 만한 방법이 없습니다.
*   **디지털 격차:** 많은 고령자는 손의 민첩성, 시력, 또는 디지털 리터러시가 부족해 스마트폰 조작이 어렵습니다. 음성은 가장 자연스러운 인터페이스이지만, 항상 손에 쥐고 항상 켜져 있는 기기가 필요합니다.
*   **인지 참여 및 정서적 지원:** 외로움과 인지 기능 저하는 고령자에게 심각한 건강 위험입니다. 수동적인 기기는 어떠한 대화 참여나 인지 자극도 제공하지 못합니다.

과제는 사용자가 이미 손에 쥐고 있는 기기인 지팡이에 이 모든 기능을 내장하되, 최종 사용자에게 기술적 복잡성을 추가하지 않는 것이었습니다.

### 솔루션: 지팡이 위의 MOTA AI Agent 플랫폼


#### 하드웨어 아키텍처

디바이스는 전용 하드웨어와 MOTA의 클라우드-엣지 소프트웨어 레이어를 결합합니다:

| 컴포넌트 | 기능 |
|---|---|
| **음성 칩** | 디바이스 내 오디오 캡처 및 재생 하드웨어 |
| **리얼다 4G 모듈** | 상시 연결 셀룰러 네트워크, 클라우드 추론 지원 |
| **베이더우 측위 칩** | 실시간 위성 GPS 위치 추적 |
| **AI 버튼** | 누르고 말하는 단일 버튼 인터페이스 — 화면도, 앱도 불필요 |

#### 상호작용 흐름: 푸시 투 토크

전체 사용자 경험은 고령자의 운동 능력에 최적화된 단일 물리적 제스처로 축소됩니다:

1. AI 버튼을 **길게 누름** → 마이크가 활성화되어 사용자가 자연스럽게 말합니다.
2. 버튼을 **놓음** → 음성 입력이 4G를 통해 클라우드 LLM으로 스트리밍됩니다.
3. LLM이 쿼리를 처리하고 합성 음성으로 응답을 스트리밍 반환합니다.
4. 사용자는 내장 스피커를 통해 응답을 듣습니다.

**인터럽트 지원:** LLM이 응답을 재생하는 도중 사용자가 버튼을 다시 길게 누르면 재생이 즉시 중단되고, 시스템은 새로운 음성 쿼리를 받을 준비를 합니다.

#### LLM 통합: 중국이동 대규모 모델

MOTA의 Agent 플랫폼에 **중국이동 LLM** 커넥터가 추가되었습니다. 기업 고객은 에이전트 설정에서 이를 기반 모델로 선택할 수 있습니다:

*   **자연스러운 대화** — 고령자에게 흔한 생략적·구어체 표현을 이해합니다.
*   **도메인별 지식** — 에이전트 구성을 통한 건강 알림, 복약 일정, 지역 날씨 등 맞춤 정보.
*   **세션 내 멀티턴 메모리** — 자연스럽게 흐르는 연속 대화 지원.

#### Agent API 레이어

MOTA는 난징 리치의 하드웨어 플랫폼을 위한 두 가지 깔끔한 통합 인터페이스를 설계했습니다:

*   **ASR (음성 인식):** 마이크로 수신한 원시 오디오 스트림을 받아 LLM 추론을 위한 텍스트를 반환합니다.
*   **TTS (음성 합성):** LLM의 텍스트 응답을 받아 합성 음성 스트림과 재생 명령을 케인 스피커로 반환합니다.

명확한 API 경계 덕분에 하드웨어 팀과 AI 플랫폼 팀이 독립적으로 발전할 수 있어 통합 복잡성이 줄고 향후 업데이트가 빨라집니다.

### 방안의 가치

음성 AI, 위성 측위, LLM 에이전트를 일상적으로 사용하는 의료 보조 기기에 통합함으로써 세 가지 차원에서 측정 가능한 가치를 제공합니다:

*   **안전 보장:** 베이더우 GPS로 가족이나 돌봄 제공자가 실시간 위치를 공유받을 수 있어 신속한 응급 대응이 가능합니다.
*   **접근성:** 버튼 하나만 누르면 됩니다 — 스마트폰도, 앱도, 학습 비용도 없습니다.
*   **정서적 동반자:** 자연어 LLM 대화는 혼자 살거나 교류가 적은 환경의 고령자에게 인지 자극과 정서적 연결을 제공합니다.
