### 시장 배경: 3000억 위안 시장의 데이터 문제

중국 펫 시장은 **2025년 3126억 위안**을 돌파하고, 2028년에는 **4050억 위안**에 달할 전망입니다. 가장 빠르게 성장하는 세그먼트는 스마트 하드웨어로, 2024년 기준 인텔리전트 펫 용품은 전체 시장의 **20%** 를 차지했으며 반려 로봇과 건강 기기의 증가세가 눈에 띕니다. 이 성장을 이끄는 주체는 젊은 세대—90년대생·00년대생 반려인이 신규 입양자의 다수를 점하고 있습니다.

하지만 폭발적인 성장의 이면에는 구조적 문제점이 존재합니다:

*   **데이터 사일로:** 각 하드웨어 브랜드가 독립적으로 운영되어 A 브랜드의 GPS 목줄과 B 브랜드의 건강 모니터 데이터가 연동되지 않습니다. 반려인은 조각난 데이터 앞에 통합된 건강 정보를 얻지 못합니다.
*   **전문 케어 비용:** 동물병원 진료는 비싸고 접근성이 낮으며, 반려인은 신뢰성이 불분명한 온라인 커뮤니티에 의존합니다.
*   **범용 AI의 정확도 한계:** 범용 LLM은 품종별 영양 관리, 종별 질병 패턴, 행동 이상 등 세부적인 펫케어 질문에 정확히 답할 도메인 지식이 없습니다.
*   **수동적인 기록만 가능:** 현재 솔루션은 기록만 할 뿐 어드바이스를 제공하지 않습니다.

### 솔루션: 하드웨어를 감지 입점으로 + 7개 독자 AI 모델

링지이의 아키텍처는 명확한 원칙에 기반합니다: **하드웨어는 AI에 데이터를 공급하기 위해 존재하며, 제품의 종착점이 아닙니다**. 세 가지 스마트 기기가 상시 가동 감지 노드로 기능하며 통합 AI 레이어에 지속적으로 멀티모달 데이터를 공급합니다.

#### 세 가지 핵심 IoT 기기

**① 스마트 포지셔닝 목줄**

모든 반려인이 가지는 핵심 불안——*내 반려동물이 돌아올 수 있을까?*——을 해소하는 안전 레이어.

*   실시간 GPS 위치 추적, 정확도 **≤ 5미터**
*   다중 전자 지오펜스 설정, 이탈 즉시 알림
*   강제 탈거 감지 경보
*   원터치 원격 벨소리 기능으로 숨어 있는 반려동물을 빠르게 찾기

**② 스마트 펫 로봇카**

반려동물 양육의 공간적 제약을 없애는 인터랙티브 이동형 동반자. 반려인이 외출 중일 때 스마트폰으로 원격 조작해 집 안을 돌아다니며 반려동물과 양방향 교감을 유지합니다. 차량 탑재 HD 카메라가 촬영한 영상·사진은 GenAI 콘텐츠 파이프라인에 직접 공급됩니다.

**③ 펫 AI 모니터**

상시 가동 건강·행동 감시 기기:

*   20개 이상 행동 패턴과 발성 신호 지속 캡처
*   배설물·구토물의 **색상·형태·질감** 이미지 분석으로 소화기 질환 및 중독 위험 조기 경보
*   화장실 이상 감지——빈도·배출량·지속 시간이 정상 범위를 벗어나면 건강 알림 발송

### 7개 버티컬 AI 모델

독자 모델 레이어가 링지이를 단순 센서 데이터 집계 플랫폼과 차별화하는 핵심 경쟁력입니다.

#### 모델 01 — 펫 특징 인식
*   **100종 이상의 고양이·개 품종** 인식, 정확도 **≥ 95%**, 이색 동물 80+ 종 지원
*   **8가지 감정 상태**(행복, 불안, 공포, 놀이 등) 및 **20+ 행동**(파괴, 그루밍, 사냥 본능, 휴식 등) 인식
*   사진 한 장으로 다차원 성격·건강 평가 리포트 생성

#### 모델 02 — 체징 비전 분석
4개 해부 부위——**눈·피부·구강·귀도**——에 컴퓨터 비전을 적용해 AI 건강 상태 리포트를 출력합니다. 배설물 분석으로 임상 증상 이전에 전신 질환의 초기 신호를 감지합니다.

#### 모델 03 — 펫 지식 Q&A (수의 LLM)
*   **100만 항목 도메인 지식베이스** 기반
*   **24시간 365일** 지능형 증상 상담 및 반려동물 케어 Q&A
*   수의 진단 로직을 시뮬레이션해 예비 평가와 케어 권고안을 생성

#### 모델 04 — 스마트 앨범 자동 분류
사진 속 반려동물을 행동(식사·놀이·수면) 또는 품종으로 자동 인식해 다두 가정의 포토 라이브러리를 수동 태깅 없이 정리합니다.

#### 모델 05 — 생성 AI 엔터테인먼트
**이미지→이미지·이미지→영상** 생성을 활용해:
*   반려동물 전용 코디 시각화
*   밈 및 스티커 팩
*   스크립트 기반 숏폼 비디오("펫 시트콤") 생성 후 SNS 원클릭 게시

#### 모델 06 — 펫 커뮤니케이션·번역
*   발성 신호를 문맥과 함께 인식·분류(상황별 울음소리, 짖음, 칭얼거림)
*   발성을 자연어 텍스트로 변환
*   반려동물의 감정 상태에 맞춘 1인칭 공감 응답을 AI가 생성해 양방향 대화를 연출

#### 모델 07 — 미아 방지 시스템
위치·행동·네트워크 데이터를 통합한 종합 안전 레이어:
*   GPS 정확도 ≤ 5m + 72시간 이동 경로 기록
*   다중 지오펜스 개별 설정
*   **실종 반려동물 상호 지원 네트워크**——실종 신고 시 인근 등록 사용자에게 일괄 알림

### 폐쇄 루프 시나리오 예시

**시나리오 A — 공원 산책 → 탈출 경보 → 신속 발견**

1. 목줄이 지오펜스 이탈 감지 → 스마트폰에 즉시 푸시 알림
2. 반려인이 양방향 오디오 활성화; 원격으로 반려동물 안정시키기
3. 반려동물 소리가 실시간으로 스마트폰에 전송
4. GPS 경로 + 원격 벨소리 기능으로 정확한 위치 파악

**시나리오 B — 혼자 집 → 건강 이상 → 반려인 알림**

1. AI 모니터가 화장실 방문 빈도 이상 감지
2. 모델 02가 캡처 이미지 분석, 배설물 색상 이상 플래그
3. 시스템이 예비 진단 컨텍스트와 함께 건강 알림 발송
4. 반려인이 알림 화면에서 바로 모델 03(수의 Q&A) 호출해 상담

**시나리오 C — 로봇카 촬영 → GenAI 제작 → SNS 게재**

1. 반려인이 원격으로 로봇카 조작, 놀이 순간 포착
2. 스마트 앨범이 자동 태그(행동: 놀이; 품종 확인)
3. 모델 05가 밈 오버레이 효과 + 배경음악 추천을 생성
4. 반려인이 원클릭으로 틱톡/샤오홍수에 게시

### AI × 중국 제조 방법론의 모범 사례

링지이의 아키텍처는 MOTA TECHLINK의 핵심 명제를 입증합니다: **버티컬 AI와 중국 하드웨어 공급망 우위의 결합이 방어 가능한 고부가가치 제품을 만들어냅니다**. 세 가지 기기는 GPS 모듈·4G 칩셋·카메라 센서 분야의 중국 원가 경쟁력을 활용하고, 7개의 독자 모델은 하드웨어 단독 경쟁자가 복제할 수 없는 차별화를 제공합니다. 플랫폼의 가치는 사용자 증가와 함께 복리로 성장합니다——행동 데이터가 쌓일수록 모델 정확도가 향상되고, 등록 사용자가 늘수록 상호 지원 네트워크가 강화되며, 소셜 콘텐츠가 증가할수록 브랜드 플라이휠이 가속됩니다.
