## 概览

搜索引擎优化 (SEO) 对于提高网站在搜索引擎上的可见性至关重要。本指南说明了如何为您的 Next.js 应用程序配置 Google Analytics 和 Google Search Console。

## 环境变量

要启用 SEO 功能，您需要在 `.env` 或 `.env.local` 文件中配置以下环境变量：

```bash
NEXT_PUBLIC_GA_MEASUREMENT_ID=G-XXXXXXXXXX
NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION=your-verification-code
```

## Google Analytics (`NEXT_PUBLIC_GA_MEASUREMENT_ID`)

Google Analytics 4 (GA4) 帮助您了解访问者如何与您的网站互动。

### 如何获取测量 ID (Measurement ID)：

1.  前往 [Google Analytics](https://analytics.google.com/) 并使用您的 Google 账号登录。
2.  点击左下角的 **管理 (Admin)** (齿轮图标)。
3.  在 **媒体资源 (Property)** 列中，点击 **数据流 (Data Streams)**。
4.  选择您的 **Web** 数据流。如果您没有，请点击 **添加数据流 (Add stream)** -> **Web** 并创建一个。
5.  在 **Web 数据流详情** 窗口中，复制 **测量 ID (MEASUREMENT ID)** (通常以 `G-` 开头)。
6.  将此 ID 粘贴到您的 `.env` 文件中，作为 `NEXT_PUBLIC_GA_MEASUREMENT_ID`。

## Google Search Console (`NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION`)

Google Search Console 工具帮助您衡量网站的搜索流量和表现，修复问题，并使您的网站在 Google 搜索结果中脱颖而出。

### 如何获取验证码 (Verification Code)：

1.  前往 [Google Search Console](https://search.google.com/search-console) 并登录。
2.  点击 **添加资源 (Add property)** (或者如果已添加，从下拉菜单中选择您的资源)。
3.  选择 **网址前缀 (URL prefix)** 方法并输入您的网站网址 (例如 `https://your-domain.com`)。点击 **继续 (Continue)**。
4.  在验证方法选项中，向下滚动到 **HTML 标记 (HTML tag)**。
5.  复制提供的元标记。它看起来像这样：
    `<meta name="google-site-verification" content="your-verification-code" />`
6.  仅提取 content 值 (`your-verification-code`)。
7.  将此代码粘贴到您的 `.env` 文件中，作为 `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION`。

### 验证配置

添加这些变量并部署您的网站 (或重启本地服务器) 后：

-   **Google Analytics**: 访问您的网站，并检查 Google Analytics 中的 **实时 (Realtime)** 报告，查看您的访问是否被追踪。
-   **Google Search Console**: 返回 Google Search Console 中的验证页面，点击 **验证 (Verify)**。它应该会确认所有权已通过验证。
