## 订购

**如何下订单？**
您可以通过任意产品页面的“联系销售”按鈕提交免费咨询，或直接在线下单。详情请参阅[如何下订单](/docs/getting-started/how-to-order)指南。

**可以申请产品页面未列出的自定义套餐吗？**
可以。请联系我们提供具体需求，我们将为您准备定制方案。

**有最低起订量吗？**
我们的入门级套餐起价 $99。标准套餐无更低最小值限制。

## 交付

**交付需要多长时间？**
交付时间因服务不同而异。网站建设入门版需 5–7 个工作日，硬件 MVP 基础版需 15–20 个工作日。详细时间表请参阅[支付与交付](/docs/services/payment-delivery)页面。

**付款后会发生什么？**
您将收到订单确认邮件，随后项目经理将在 1 个工作日内联系您安排启动会议。

**交付后如需修改怎么办？**
所有套餐均包含免费支持期（根据方案为 1–6 个月）。在此期间，错误修复和小幅调整均免费。

## 支付

**接受哪些支付方式？**
我们接受 Visa/Mastercard（通过 Stripe）、USDC、ETH 和 BTC。详情请参阅[支付与交付](/docs/services/payment-delivery)。

**信用卡支付安全吗？**
是的。所有卡支付均通过 PCI-DSS 1 级认证的 Stripe 处理。

**可以用本地货币支付吗？**
我们的标价为美元。Stripe 会自动处理卡支付的货币换算。

**什么是 X402？**
X402 是 Base 区块链上的 HTTP 原生微支付协议。我们用它实现每次 $0.10 USDC 的 AI 即时咨询。使用时需要 MetaMask 钱包。

## 技术

**您建设的网站支持移动端吗？**
是的。我们所有的网站均完全响应式，并在 iOS 和 Android 上经过测试。

**交付后我可以自行更新网站内容吗？**
是的。我们会移交 Git 仓库，并提供如何编辑内容文件和重新部署的指导。

**多语言网站支持哪些语言？**
默认支持英文（EN）、中文（ZH）、日文（JA）和韩文（KO）。可另行安排支持其他语言。

## 支持

**如何联系支持？**
通过[联系](/contact)页面发送邮件给我们。响应时间在 1 个工作日内。

**提供持续维护合同吗？**
是的。免费支持期结束后，我们提供月度维护计划。请联系我们了解定价。
