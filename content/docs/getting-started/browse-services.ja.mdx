## 当サイトの歩き方

MOTA TECHLINK のサイトは、サービスの決定からお問い合わせまでをスムーズに進められるよう設計されています。このページでは、各セクションの概要をご紹介します。

## メインナビゲーション

全ページの上部にメインナビゲーションバーがあり、以下のセクションが含まれています：

- **プロダクト** — MOTA の全サービス：Web サイト構築、ハードウェア MVP、AI チャットボット、スケールアップ生産。
- **リソース** — 導入事例、ブログ記事、ヘルプセンター、ドキュメント。
- **料金** — サービスプランと開始価格の概要。
- **会社情報** — 会社概要、お問い合わせ先、法律ページ。

## サービスの探し方

### プロダクトページ

[/products](/products) で全サービスを閲覧できます。各カードには以下が表示されます：
- サービスの簡潔な説明
- 開始価格
- 使用技術
- 詳細ページへのリンク

### サービス詳細ページ
各サービスをクリックすると、専用詳細ページが開きます：

1. **ギャラリー** — 過去の納屋事例のスクリーンショットや対応写真。
2. **プラン比較** — Basic / Standard / Premiumプランの機能一覧。
3. **サービスについて** — 納品内容と進行方法の詳細説明。
4. **使用技術** — 各サービスで使用する技術スタック。
5. **よくある質問** — そのサービスに特化した FAQ。
6. **導入事例** — 実際に納品した関連ショーケースプロジェクト。

## ショーケース（導入事例）

[/showcase](/showcase) で完成プロジェクトを閲覧できます。

- カテゴリーで絞り込み（AI、ハードウェア、Webなど）
- キーワード検索
- カードをクリックして演円な導入事例を読む

## ブログ

[/blog](/blog) で、AI・ハードウェア・ビジネス成長に関する最新記事をお読みいただけます。記事は四つの言語（EN / ZH / JA / KO）でご利用いただけます。

## ヘルプセンター

[ヘルプセンター](/help)には MOTA の RAG 技術を搭載した AI アシスタントがご利用いただけます。サービスに関するご質問は何でもお気軽にドうぞ。

## 言語切り替え

全ページは英語・中文・日本語・韓国語の四言語に対応しています。ページ上部の言語切り替えボタンからいつでも言語を変更できます。
