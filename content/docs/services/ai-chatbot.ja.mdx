## 概要

AI チャットボットサービスは、あなたのウェブサイトやプラットフォームに検索拡張生成（RAG）チャットボットを導入します。チャットボットは製品ドキュメント、FAQ、ビジネスコンテンツから学習し、24 時間 365 日多言語で訪問者の質問に即時回答します。

汎用 AI アシスタントとは異なり、当チャットボットは*あなたの*ナレッジベースに基づいているため、回答は正確でブランドに沿い、常に最新の状態に保たれます。

## 料金プラン

| プラン | 価格 | 適用シーン |
|--------|------|------------|
| **スターター** | $99 | 小規模サイト、基本的な Q&A |
| **ビジネス** | $2,500 | プロダクト会社、EC |
| **エンタープライズ** | $10,000+ | 大規模組織、複数部門 |

### スターター — $99

- プレビルドのチャットボットウィジェットをウェブサイトに埋め込み
- 最大 50 ページのナレッジベース
- 既存サイト（任意フレームワーク）との統合
- 英語 + 1 言語追加
- 納品後 1 ヶ月のサポート

### ビジネス — $2,500

- 全ドキュメントに基づくカスタム RAG モデルのトレーニング
- 最大 500 ページのナレッジベース
- 多言語（最大 4 言語）
- エスカレーションフロー（人間エージェントへの引き継ぎ）
- 分析ダッシュボード（質問トラッキング、満足度レート）
- 納品後 3 ヶ月のサポート

### エンタープライズ — $10,000+

- 専用 AI チャットボットインフラ
- 無制限ナレッジベースサイズ
- CRM/ERP との完全 API 統合
- カスタム LLM ファインチューニングオプション
- X402 プロトコル AI エージェント決済処理
- SLA サポートと保証された応答時間

## 主な機能

- **RAG アーキテクチャ** — 回答生成前にドキュメントから関連コンテキストを取得し、ハルシネーションを大幅に削減。
- **多言語** — 訪問者の言語（日英中韓など）で自動的に返答。
- **24 時間 365 日稼働** — ダウンタイムなし、祝日なし、即時応答。
- **X402 統合** — プレミアム AI インタラクションのためのオンチェーンマイクロペイメント対応（例：コンサルテーション 1 回あたり $0.10 USDC）。
- **簡単な統合** — ウィジェットとして埋め込むか REST API 経由で接続。

## はじめ方

1. [AI チャットボット製品ページ](/mota-ai/chatbot)で詳細を確認。
2. **営業に問い合わせる**をクリックして、ナレッジベースと要件を相談。
3. キックオフ後、チームがデータ取り込み、モデルトレーニング、デプロイを担当します。
