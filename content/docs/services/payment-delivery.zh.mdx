## 支付方式

### 信用卡

我们通过 **Stripe** 接受 Visa 和 Mastercard，这是一家符合 PCI-DSS 标准的支付处理商。

- 通过 Stripe Checkout 安全处理付款。
- 付款后立即收到 Stripe 付款收据。
- 支持货币：美元（主要），其他货币自动换算。

### 加密货币

我们接受以下加密货币支付服务费用：

| 币种 | 网络 |
|------|------|
| USDC | Base、以太坊 |
| ETH | 以太坊 |
| BTC | 比特币 |

加密货币付款说明将在结账时提供。所有加密货币交易均为最终交易，不可撤销。

### X402 链上支付（即时咨询）

针对快速咨询功能，我们使用 **X402 协议** — 一种基于 Base 网络的 HTTP 原生微支付标准。

- 连接您的 MetaMask 钱包。
- 每次咨询支付 $0.10 USDC。
- Base 网络 Layer 上无 Gas 费（交易成本极低）。
- 付款自动处理，AI 回复即时解锁。

## 交付流程

1. **订单确认** — 付款后立即收到确认邮件。
2. **启动会议** — 项目经理在 1 个工作日内联系您确定需求。
3. **开发与更新** — 我们在关键里程碑发送进度更新邮件。
4. **审查与修改** — 正式交付前，您将收到预览链接（网站）或审查会议（硬件/AI）。
5. **最终交付** — 凭据、文件或实物运输交付给您。

## 售后支持

所有套餐均包含最低保修期：

- **入门版套餐**：1 个月免费故障修复支持。
- **标准版套餐**：3 个月免费故障修复支持。
- **高级版套餐**：6 个月专属支持。

支持通过邮件提供，响应时间：1 个工作日内。

## 退款政策

- **启动前**：下单后 24 小时内可全额退款。
- **启动后**：根据已完成工作部分退款，请联系我们。
- **加密货币付款**：链上确认后不可退款（如有疑虑，请在下单前联系我们）。
