## 개요

AI 챗봇 서비스는 귀하의 웹사이트나 플랫폼에 검색 증강 생성(RAG) 챗봇을 배포합니다. 챗봇은 제품 문서, FAQ, 비즈니스 콘텐츠에서 학습하여 24시간, 다국어로 방문자 질문에 즉시 응답합니다.

일반 AI 어시스턴트와 달리, 당사 챗봇은 *귀하의* 지식 베이스를 기반으로 하므로 응답이 정확하고 브랜드에 맞으며 항상 최신 상태입니다.

## 가격 플랜

| 플랜 | 가격 | 적합한 용도 |
|------|------|------------|
| **스타터** | $99 | 소규모 사이트, 기본 Q&A |
| **비즈니스** | $2,500 | 제품 회사, 이커머스 |
| **엔터프라이즈** | $10,000+ | 대규모 조직, 다부서 |

### 스타터 — $99

- 웹사이트에 임베드된 사전 구축 챗봇 위젯
- 최대 50페이지 지식 베이스
- 기존 사이트와 통합 (모든 프레임워크 지원)
- 영어 + 1개 추가 언어
- 납품 후 1개월 지원

### 비즈니스 — $2,500

- 전체 문서 기반 맞춤형 RAG 모델 학습
- 최대 500페이지 지식 베이스
- 다국어 (최대 4개 언어)
- 에스컬레이션 플로우 (담당자 연결)
- 분석 대시보드 (질문 추적, 만족도)
- 납품 후 3개월 지원

### 엔터프라이즈 — $10,000+

- 전용 AI 챗봇 인프라
- 무제한 지식 베이스
- CRM/ERP 완전 API 통합
- 맞춤형 LLM 파인튜닝 옵션
- X402 프로토콜 AI 에이전트 결제 처리
- SLA 지원 및 보장된 응답 시간

## 주요 기능

- **RAG 아키텍처** — 응답 생성 전 문서에서 관련 컨텍스트를 검색하여 환각을 대폭 감소.
- **다국어** — 방문자의 언어(한/영/중/일 등)로 자동 응답.
- **24시간 가용성** — 다운타임 없음, 휴일 없음, 즉시 응답.
- **X402 통합** — 프리미엄 AI 인터랙션을 위한 온체인 마이크로 결제 지원 (예: 상담 1회당 $0.10 USDC).
- **간편한 통합** — 위젯으로 임베드하거나 REST API로 연결.

## 시작하는 방법

1. [AI 챗봇 제품 페이지](/mota-ai/chatbot)에서 자세한 내용을 확인하세요.
2. **영업팀 문의**를 클릭하여 지식 베이스와 요구사항을 논의하세요.
3. 킥오프 후 팀이 데이터 수집, 모델 학습, 배포를 담당합니다.
