## 支払い方法

### クレジットカード

PCI-DSS 準拠の決済処理業者 **Stripe** を通じて、Visa と Mastercard をご利用いただけます。

- Stripe Checkout を通じて安全に決済処理。
- 支払い後すぐに Stripe 領収書を受け取ります。
- 対応通貨：USD（主要）、他の通貨は自動換算。

### 暗号通貨

以下の暗号通貨でサービス料金をお支払いいただけます：

| 通貨 | ネットワーク |
|------|------------|
| USDC | Base、Ethereum |
| ETH | Ethereum |
| BTC | Bitcoin |

暗号通貨の支払い手順はチェックアウト時に提供されます。すべての暗号通貨取引は最終的なものであり、取り消し不可です。

### X402 オンチェーン決済（即時コンサルティング）

クイックコンサルティング機能には **X402 プロトコル** を使用しています — Base ネットワーク上に構築された HTTP ネイティブのマイクロペイメント標準です。

- MetaMask ウォレットを接続します。
- コンサルティングクエリ 1 回あたり $0.10 USDC をお支払いください。
- Base ネットワークのレイヤーでガス代なし（最小限のトランザクションコスト）。
- 決済は自動処理され、AI 回答が即座にアンロックされます。

## 納品スケジュール

| サービス | プラン | 納期 |
|---------|--------|------|
| ウェブサイト制作 | スターター | 5〜7 営業日 |
| ウェブサイト制作 | スタンダード | 2〜3 週間 |
| ハードウェア MVP | ベーシック | 15〜20 営業日 |
| ハードウェア MVP | スタンダード | 3〜5 週間 |
| AI チャットボット | スターター | 3〜5 営業日 |
| AI チャットボット | ビジネス | 1〜2 週間 |
| 全サービス | プレミアム/カスタム | 合意による |

営業日は月曜日〜金曜日、香港と中国本土の祝日を除きます。

## 納品プロセス

1. **注文確認** — 支払い後すぐに確認メールが届きます。
2. **キックオフコール** — プロジェクトマネージャーが 1 営業日以内に連絡し、要件を確認します。
3. **開発と更新** — 主要なマイルストーンでの進捗更新メールをお送りします。
4. **レビューと修正** — 最終納品前に、プレビューリンク（ウェブサイト）またはレビューセッション（ハードウェア/AI）を提供します。
5. **最終納品** — 認証情報、ファイル、または物理的な発送があなたにお届けされます。

## 納品後サポート

すべてのパッケージには最低保証期間が含まれます：

- **スタータープラン**：1 ヶ月間の無料バグ修正サポート。
- **スタンダードプラン**：3 ヶ月間の無料バグ修正サポート。
- **プレミアムプラン**：6 ヶ月間の専任サポート。

サポートはメールで提供されます。応答時間：1 営業日以内。

## 返金ポリシー

- **キックオフ前**：注文後 24 時間以内は全額返金可能。
- **キックオフ後**：完了済み作業に基づく部分返金；お問い合わせください。
- **暗号通貨決済**：オンチェーンで確認後は返金不可（懸念がある場合は注文前にご連絡ください）。
